的“符合执业规范”的那篇微博开始，了解了很多事情后才怼上的，在<a href="http://weibo.com/ttarticle/p/show?id=2309404029136959089106" target="_blank">我为何如此关注杨永信</a>里说了些，不多说了。<br/><br/>发了这邮件算是告一段落，不能再花太多精力在杨永信这件事上了，因为到了挺关键的时期，之后估计只能转发下微博、刷刷热度，虽然之前也是这样，但花的时间会少很多很多。<br/><br/>写了意见信才发现自己很久很久没写过正式的书信了，现在写初中生的作文估计都写不出来，遣词造句、疏通逻辑还得练练。<br/><br/>意见信如下：?<br/><br/>?<br/><br/><hr /><br/><br/>??<br/><br/>网信办的同志：<br/><br/>你好。<br/><br/>本人对《未成年人网络保护条例（草案征求意见稿）》的第二章第十二条、第四章第十九条、第二十条、第二十三条持有意见，具体意见如下：<br/><br/>第十二条 智能终端产品制造商在产品出厂时、智能终端产品进口商在产品销售前应当在产品上安装未成年人上网保护软件，或者为安装未成年人上网保护软件提供便利并采用显著方式告知用户安装渠道和方法。<br/><br/>意见：支持保护软件的开发，反对在所有只能终端设备上强制安装。<br/><br/>1.? 智能终端的消费人群包括未成年人和成年人，成年人不应当受未成年人上网保护软件的干预。<br/><br/>2.? 消费者拥有自主下载、安装和使用未成年人上网保护软件的权利。<br/><br/>3.? 在个人产品中强制安装未成年人上网保护软件，侵犯消费者自主选择权，违反《中华人民共和国消费者权益保护法》。<br/><br/>4.? 强制安装未成年人上网保护软件，将构成强制交易，同样违反《中华人民共和国消费者权益保护法》。<br/><br/>5.? 市面上的智能手机、计算机系统中，已有相似的功能，如华为的手机系统、Windows7和Windows10中，均有未成年人保护的功能，类似的软件和功能可以由商家提供，并为家长的使用提供便利，但不应该强制安装某一种软件，形成不正当竞争，违反《中华人民共和国反不正当竞争法》。<br/><br/>6.? 随着智能设备的发展，个人隐私、信息安全日益受到关注，未成年人上网保护软件将可能侵犯个人隐私，泄露个人信息，强制安装指定的未成年人上网保护软件，公民将无法规避该风险，这将会影响政府的公信力，参考绿坝软件。<br/><br/>第十九条 家庭、学校应当教育、引导未成年人正确使用网络，预防和干预未成年人沉迷网络。对于有沉迷网络倾向的未成年学生，学校应当指导其监护人开展家庭教育，配合家庭、社区及其他机构进行教育和引导。<br/><br/>意见：“其他机构”过于模糊，难以保护未成年人合法权益。<br/><br/>1.? 未成年人沉迷网络本质上并非“精神疾病”，可能由家庭教育失职或未成年人心理问题导致，需要由父母、孩子与第三方正规心理咨询机构进行合作，以帮助教育、引导和干预未成年人正确使用网络。学校并不具备专业资质进行心理咨询和干预。<br/><br/>2.? “其他机构”需要具备什么样的资质，这里并没有写明确；目前市场上的“网瘾戒治”机构、学校等，均无国家资格认定，此类学校多以军训和传统文化教育作为宣传，但近十年来，造成的致死致残事故不下百起，这类机构会对未成年人的健康造成不可预料的伤害。相关案件如下：<br/><br/>近期，山东科技防卫专修学院“设套”抓学生、暴力体罚等乱象被曝光在公众面前；2009年，一名网瘾少年在被送到南宁市一家“南宁起航拯救训练营”10多个小时后死亡；2014年5月19日，“郑州搏强新观念培训学校”一位女生因颅脑损伤而死亡；2014年4月12日，一位14岁少年在“云南曲靖沾益县焕然成长训练中心”被体罚，司法鉴定为重伤二级、伤残八级。还有其他许多类似的网戒学校发生的案件。<br/><br/>建议：<br/><br/>1.? 明确“其他机构”为何种机构，明确其需具备何种资质，并对相关机构进行严格审查，对其权力范围进行严格控制，发现未成年人遭受该类机构侵害时，应当及时采取措施予以救助，必要时向公安机关或其他有关部门举报，接到举报的单位应及时受理。<br/><br/>2.? 教育、引导未成年人正确使用网络，应由专业人士进行介入，并对其资质需要进行严格审查；在开展教育时，需在当事人具有行为能力、本人愿意及监护人同意的前提下，进行教育和引导，但不可对未成年人不涉他行为进行实质性的干预，强制介入和干扰未成年人的个人生活。<br/><br/>第二十条 教育、卫生计生等部门依据各自职责，组织开展预防未成年人沉迷网络的教育，对未成年人网络成瘾实施干预和矫治。<br/><br/>县级以上人民政府及其教育行政部门应当为中小学校配备专门教师或提高教师专业水平，增强对未成年人沉迷网络的早期识别和干预能力，并可以通过购买服务等方式提供预防和干预未成年人沉迷网络的教育或服务。<br/><br/>国务院卫生计生部门会同有关部门推动出台网络成瘾的本土化预测和诊断测评系统，制定诊断、治疗规范。<br/><br/>意见：反对“对未成年人网络成瘾实施干预和矫治”，支持对未成年人沉迷网络进行教育和引导。<br/><br/>1.? “网络成瘾”目前并未被确认为一种疾病，“网络成瘾尚缺乏发病率、病程、生物因素、遗传性等重要的流行病学信息，需要在这方面进一步研究才能确认它是一种疾病”，网络成瘾更多是由于家庭教育等原因造成的一种现象。卫生部《未成年人健康上网指导》征求意见稿中提出：“讨论认为，目前‘网络成瘾’定义不明确，不应以此界定不当使用网络对人身体健康和社会功能的损害”。<br/><br/>2.? 因目前尚未被定性为精神疾病，故不支持“推出网络成瘾的本土化预测和诊断测评系统，制定诊断、治疗规范”，建议将“网络成瘾”改为“网络使用不当”，对其进行教育和引导，反对强制干预和介入。<br/><br/>3.? 反对“配备专门教师”，教师是否有专业资质对学生的生活进行介入和干预？建议专门的教师进行心理咨询，并提供帮助，但不可以介入未成年人的个人生活。<br/><br/>4.? 同第十九条的“其他机构”，目前此类机构的资质存在问题，且发生过许多起恶劣事件。故反对“购买服务”的方法对未成年人的个人生活进行干扰和强制介入。<br/><br/>第二十三条 网络游戏服务提供者应当建立、完善预防未成年人沉迷网络游戏的游戏规则，对可能诱发未成年人沉迷网络游戏的游戏规则进行技术改造。<br/><br/>网络游戏服务提供者应当按照国家有关规定和标准，采取技术措施，禁止未成年人接触不适宜其接触的游戏或游戏功能，限制未成年人连续使用游戏的时间和单日累计使用游戏的时间，禁止未成年人在每日的0:00至8:00期间使用网络游戏服务。<br/><br/>意见：“技术改造”过于模糊，实现难度较大。<br/><br/>建议：建立对网络游戏等的分级制度，对“可能诱发未成年人沉迷网络游戏的游戏规则”进行详细规定，详细说明技术改造的具体内容。<br/><br/>以上是我对该草案征求意见稿的部分条例提出的意见与建议。<br/><br/><span
