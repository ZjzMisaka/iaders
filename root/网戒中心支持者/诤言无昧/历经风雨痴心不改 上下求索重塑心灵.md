<br/>―――写在临沂网戒中心成立七周年之际<br/><br/>本报特约记者岳增学<br/><br/>编者按：一元复始，万象更新。 励志・孝道・爱心教育在这辞旧迎新之际，备受社会关注的临沂网戒中心也迎来了自己的七周年华诞。七年来，面对给社会带来严重危害、给家庭带来无尽伤痛的网瘾之患，临沂网戒中心勇担社会道义，敢为人先，成功创建了史无前例的网戒事业。因为是新生事物，所以他们肩负使命，心系民生，在社会各界正义的呼声中，百折不挠，上下求索，不辱使命，收获累累硕果，为国为民交上了一份圆满的答卷，得到了社会的高度认可。<br/><!--more--><br/><br/><strong>互联网，后现代社会的时尚与时殇</strong><br/><br/>“现在一上课，打盹睡觉的学生越来越多，大多是上网吧熬夜所致，有些甚至已经发展到夜不归宿、极端叛逆，网瘾已经成为困扰学校和家庭教育的潜在威胁，网瘾之害猛于虎啊！”<br/><br/>一些资深教师忧心忡忡地说。著名网戒专家、临沂市第四人民医院副院长、网戒中心主任杨永信指出，总是想着去上网，不能上网时会感到烦躁不安、情绪低落或无所适从，觉得在网上比在现实生活中更快乐或更能实现自我，可以认定为网瘾倾向。<br/><br/>互联网的普及，无疑是人类历史上的一次伟大革命，然而互联网也是利益与灾难共存的双刃剑。中国卫生部于2009 年把网瘾定义为一种广义的精神疾病，网瘾少年由最初的痴迷网络迅速演变升级：心理扭曲、性格改变、逃学厌学、离家出走、早恋堕落、偷抢拐骗、嗜赌涉毒、杀父弑母的报道屡屡曝光。<br/><br/>2000 年底，中国网民有2250 万。2001 年，美国网民为1 . 65 亿。2008 年2 月，仅我国网民数达2 . 21亿人，超过美国居全球首位。截至2012 年6 月底，据工业和信息化部电信管理局公布的《第30 次中国互联网络发展状况统计报告》显示，中国网民达到5 . 38 亿，初中学历人群占比继续保持增长。中国青少年网络协会发布的《2011 年中国网络青少年网瘾调查数据报告》显示， 2011 年我国网络青少年网瘾的比例高达26% ，网瘾倾向比例高达12% ，全国大约有2400 万网瘾青少年。<br/><br/>网络成瘾已经成为严重的社会问题，而网瘾戒治，业已成为社会不能回避的责任担当。<br/><br/><strong>临沂网戒中心，乱云飞渡中的大气从容</strong><br/><br/>网瘾作为公认的后现代文明之殇，网戒也必然成为举世期待的新生事物。因为新生，她缺乏世界统一认可的概念界定、治疗标准和治疗方法，她诞生的第一天起，就在备受争议中艰难地成长。在这个过程中，杨永信和他的临沂网戒中心注定经历一个浴火涅的历程。<br/><br/>1995 年，美国精神病学家戈登伯格提出“网络成瘾症”一说，此后学界便没有停止过对它的争议，从网络成瘾概念到网络成瘾机制或依赖物质的争论，从有无网络成瘾的争议到网络成瘾诊断标准的争论…… 2004 年，荷兰人史密斯和琼斯在阿姆斯特丹开办了欧洲首家网瘾诊治所，两年后宣布失败。2005 年，北京未成年人保护委员会与中国科学院心理研究所等数家机构，从100 多个报名者中选出13 名“深度网瘾患者”受训一周，结果没有一个成功的案例。<br/><br/>2000 年开始，时任临沂市精神卫生中心心理咨询中心主任和心理康复病区主任的杨永信不断接触到一 些特殊的患者。一个花季少女，因迷恋网络，竟当众抽打父亲多记耳光；一个尚未成年的孩子，因为上网成瘾，三次在父亲的茶水饭菜里下鼠药；一名正在读高二的学生，因为网络成瘾而辍学，并发展到六亲不认，打骂父母；一名网瘾孩子纠结五百余名未成年人，打架群殴，偷窃抢劫……孩子们心理的极端扭曲，家长们痛哭流涕的绝望，唤起了杨永信勇担道义的使命感，他先后走进上百所学校做心理健康教育报告，讲述网络成瘾的危害与防治策略，借助报刊、电台、电视等媒体就网络成瘾进行“专家释疑”、“专家支招”、“专家访谈”，多次策划实施以“健康文明上网，拒绝网络成瘾”等为主题的大型社会活动，逐渐产生了越来越大的社会影响。<br/><br/>随着网瘾现象的不断加剧和杨永信知名度的不断提高，越来越多的求助者让杨永信深刻意识到网瘾问题的严重性。2006 年1 月10 日，“临沂市网络成瘾戒治中心”在全国率先挂牌成立，市级各新闻媒体也同时发布了“杨永信向网瘾宣战”的新闻。<br/><br/>从此，作为一名资深的精神卫生专家，杨永信以高度的职业责任和素养，艰难地开创着中国的网戒事业，根据对患者病理成因的遗传学、病理学、社会学、心理学、教育学的全面分析，他首先把网瘾界定为心理疾病，率先提出网瘾致病三因素理论，以“药物+心理+物理+工娱”作为探索和完善网瘾戒治模式的思维主线，从矫正网瘾患者的性格缺陷入手，把家长纳入同步诊疗对象，把更多的问题孩子和问题家庭纳入同步治疗范围，网戒中心也并列挂上了“性格缺陷矫治中心”、“青少年危险行为干预中心”、“问题家庭诊疗中心”的牌子，逐步形成了一套包含百余环节，从生理、心理到病理，从医疗、教育到家庭，从个体、群体到社会―――“三维一体”特色网瘾戒治和性格缺陷矫治模式。<br/><br/>短短的3 年时间，临沂网戒中心累计成功治愈了来自国内外的患者3000 余例，这一举世瞩目的成就，引起了央视及全国数百家媒体的高度关注和赞誉。央视1 套《法治在线》栏目2 集专题片《网梦醒来》，央视12 套《第一线》栏目12 集专题片《战网瘾・战网魔》，央视10套、9 套《人物》栏目《网瘾戒治专家杨永信》，央视1 套《百姓故事》栏目《网瘾背后》，央视内参片《谁来拯救网瘾孩子》等几十部专题片相继播出，作家出版社出版的长篇纪实报告文学《战网魔》，《光明日报》、《人民日报》等报刊的纪实性报道等，在全国引起反响，许多网瘾孩子家长从全国各地甚至国外慕名而来，杨永信成为拯救网瘾孩子的“大救星”。<br/><br/>杨永信也因此获得了享受国务院政府特殊津贴专家、山东省预防青少年违法犯罪先进个<br/><br/>人、首届山东省未成年人保护杰出公民、感动临沂年度人物、临沂十大新闻人物、山东省未成年人思想道德建设先进个人、首届感动山东十大健康卫士、山东年度新闻人物提名奖、山东省促进未成年人思想道德建设重大宣传典型、山东省“富民兴鲁”劳动奖章、山东省卫生系统“两好一满意”示范标兵、山东省第四届“发明创业奖”、山东省道德模范候选人等数十项荣誉称号或奖项，两次被山东省人社厅、卫生厅授予“二等功”，并荣登“中国好人”榜，中共临沂市委宣传部在全市举办了杨永信事迹巡回报告会。<br/><br/>没有预料到的是平静的湖面，却孕育惊涛骇浪的暗涌。2008 型治疗仪，这个机器我90 年代初期在英国作为访问学者回国后，开展专项研究的时候也用过，经过我们课题组的专项科研实际使用证明我们使用的这种仪器是安全有效的。”<br/><br/>大浪淘沙露峥嵘，乱云飞渡仍从容。一个真正代表历史发展趋势的新生事物是不可战胜<br/><br/>的。全国的网戒机构由起始阶段的3 家，高峰阶段的400 余家，到目前尘埃落定后的十余家，杨永信和他的网戒事业最终在正义的呐喊中被保护下来，并日渐显露出勃勃生机。<br/><br/>在历经三年争议后的今天，来此求助的网瘾孩子家长更加络绎不绝。2012 年初，《家庭》杂志刊发了记者杨立平的深度新闻调查，报道了网瘾给家庭带来的伤害，肯定了杨永信网瘾戒治模式的有效性和安全性。<br/><br/>据悉，目前为止，临沂网戒中心已成功救治含网瘾在内的问题孩子5000 余例，使相应的5000 多个问题家庭重获幸福。<br/><br/><strong>透析病理成因，探索安全有效模式</strong><br/><br/>杨永信的成功，在于他创新的勇气和底气。<br/><br/>他的勇气在于担当道义和挑战传统的决心；他的底气来源于专业素养的境界和把握全局的高度，对传统心理治疗理念的颠覆和网瘾治疗体系的创新构建。患者家长对医院的理性选择似乎是对杨永信和临沂网戒中心最好的认可。<br/><br/>小阳， 23 岁，同济大学在校生，因痴迷网络、学业无法继续，学校打算劝其退学，并电话告知了远在贵阳、身为某机关单位一把手的小阳爸，小阳爸在多次电话联系未果、儿子杳无音讯的情况下奔赴上海，并在学校周围的网吧里见到了形容憔悴、乱发过肩的儿子，他苦口婆心规劝，不惜向儿子下跪恳求，甚至以跳黄浦江要挟，但儿子不为所动。国庆节后，走投无路的家长把小阳“骗”到这里―――<br/><br/>杨永信：“小阳爸爸，你身为公职人员，为什么把孩子骗到这里？”<br/><br/>“因为我相信只有这里才能拯救我的孩子，还我一个美满的家庭。孩子痴迷网络，性格扭曲，根本不承认自己有病，不骗咋能送到这里来？”<br/><br/>“北京、上海甚至各大城市都有资深专家和心理咨询师，我这里只是个小地方。”<br/><br/>“那些专家一个小时的咨询费少则一千，多则数千，我们多次为孩子请名家做心理诊疗，<br/><br/>耗资巨大，效果短暂，甚至可以说无效。相比之下，临沂网戒中心价格合理，不仅治网瘾，而且治性格，不仅治孩子的病，还治家长的病，不仅局限于住院治疗，更难得的是离院后的长效机制，性格缺陷矫治是戒治网瘾的基础，父母问题是孩子问题的根，长效机制是孩子不再偏的保障。我在网上查阅了相关的资料和媒体报道，无论治疗模式还是治疗效果，这里都是全国最好的。”<br/><br/>“那你就没看到有媒体说我们这里有电击治疗，网络游戏里把我丑化为‘羊角兽’？”<br/><br/>“低剂量电刺激只是治疗的一种辅助手段，多年的实践证明了它的安全可靠，而网络游戏里把您丑化为‘羊角兽’，恰恰反证了您治网瘾的卓越成效。”<br/><br/>……<br/><br/>晓旭，今年大学毕业，高考前因升学压力和与家长的冲突患上了癔症，病情好转后考上大学，期间因性格缺陷屡受伤害，导致严重自闭，社会交际功能严重缺失。家人和朋友多次劝说给孩子治病，可晓旭爸一直未表态，因为他明白，孩子病在性格，根在家长，这个世界上还没有听说有治性格缺陷和家庭问题的医院。<br/><br/>孩子大学毕业不能走向社会，这是任何一个负责任的家长所不能坐视的，晓旭要么关在自己的房间整天不出门，要么早上出去，夜不归宿，打电话不接，发短信不回。带着“有枣无枣打一杆，死马当作活马医”的想法，晓旭爸通过有关专家介绍，把孩子送到网戒中心。国庆晚会，晓旭一曲深情而专业的《我爱你，中国》，激起台下经久不息的掌声；感恩节演讲，长期失语的晓旭发表了思路清晰的演说，获得了一等奖。<br/><br/>永信模式，这种以重塑心灵、健全人格，改变自我、打造完美为目标的问题青少年诊疗体系业已成为学校教育、家庭教育、社会教育的有效补充和依托。<br/><br/><strong>铁肩担道义，事实胜雄辩</strong><br/><br/>杨永信说：“临沂网戒中心成立7 求神拜佛或请过巫婆。<br/><br/>―――杨永信：我们所使用的“电刺激治疗”工具，只是利用微弱电流对个别特别狂躁的网瘾患者进行有效的情绪镇定，进而配合完成网瘾戒治综合干预模式的全过程。卫生部叫停的原因，主要还是考虑到“电击”治疗手段的技术安全性，而在多年的临床治疗中，我们还没有发现“电击”治疗产生的伤害病例，更没有收到家长在安全方面的反映或投诉。这些患者中，单纯痴迷上网的仅是极少一部分，而因痴迷网络所引发的各种危险行为(逃学厌学、离家出走、仇视父母、结伙打架、偷摸抢骗、吸毒贩毒等)、共病症状(焦虑、抑郁、强迫、幻觉、妄想等)和身体的严重损害才是让父母不惜一切代价的真正原因“。<br/><br/>每天清晨， 名的好成绩，还担任班级体育委员、年级形象标兵、校篮球队员，连续被评为三好学生。小龙的改变是得益于杨叔模式，我们全家人感恩杨叔，感恩网戒中心！”<br/><br/><strong>2 、调正人生轨迹，优化生命品质</strong><br/><br/>小伟，临沂人。走偏时一度成为街混头儿，对于其不断升级的惹是生非，父母生不如死，家庭生活暗无天日。来到网戒中心后，经过相对足疗程的住院治疗和离院四年坚持不懈的长效固化，今年以优异的成绩考入中国刑事警察学院，并以优秀的品行被任命为班长。返院分享的小伟在大点评课上一身戎装，英姿勃发，分外抢眼，杨叔对他的改变做了高度评价，成为全体盟友的标杆。<br/><br/><strong>3 、矫治性格缺陷，突破生命定律</strong><br/><br/>小超，临沂人。性格先天不足，走偏时行为怪异，令路人见之唯恐避之不及，父母求神拜佛，打卦算命，改家宅迁祖坟，天天如临末日。<br/><br/>入院很长一段时间，遍地捡拾垃圾，并收藏在病房中，搞得网戒中心人人不安，很多人并不看好他的改变前景。但杨永信与点评师们锲而不舍，辨证施治、矫正固化，现在已成功走向社会，备受单位领导和同事们的肯定。每个月发下工资，都会如数交到父母手中。父母面对重生的孩子，对杨永信感激涕零，其真诚至深无以言表。<br/><br/><strong>4 、唤醒人性良知，浪子重享天伦</strong><br/><br/>建伟，菏泽市人。走偏时迷恋网上赌博，短短几年，挥霍了父母的百万家财，令妻子长期有家不回，逼得满头白发的老父亲为亲家父母下跪。来院时其父老泪横流：“男人膝下有黄金，我活了大半辈子，除了父母祖宗，我没有给任何人下跪，可面对离家的儿媳和伤心欲绝的亲家，我下跪了，就是为了建伟这个禽兽不如的儿！”今天的建伟早已今非昔比，当杨永信问建伟“你媳妇现在回家了吗，你的家人好吗？”时，建伟响亮地回答：“报告杨叔，不是媳妇回家了，而是我回家了；不是家人好了，而是因为我好了，一切就都好了！”<br/><br/><strong>5 、融化心灵冰山，家庭幸福美满</strong><br/><br/>岩铮，河北唐山市人，国家公职人员。常年沉迷黄色网站，抛妻弃子，荒废事业，公然把过路女孩强拉回家，模仿变态的色情行为。妻子伤心欲绝，单位领导不满，身为军人出身的父亲和长期在公安工作的母亲使出浑身解数也无可奈何。来到网戒中心后，在“相亲相爱一家人”的浓郁氛围的感染下，在心理课堂和道德讲堂震撼人心的气场中，岩铮的人性开始回归，心灵恢复知觉。三个月后，岩铮以其改变过程中的出色表现，被家委任命为同盟班长，对盟友的改变做出了表率。这次分享他与爱妻共同“回家”，分享课上，美丽贤惠的岩铮媳妇对丈夫成功改变后爱妻爱子爱工作、孝敬爹妈尊他人的出色表现赞赏有加，笑声中飞扬着泪水，话语中洋溢着喜悦，岩铮对爱妻的一句真情表白“我爱你”，更是迎来了经久不息的掌声……<br/><br/>在临沂网戒中心成立七周年前夕，一位书法家给杨永信挥椽题字：“功德无量”。晓旭爸心中有感，为字作注：“公元2006 年1 月10 日，岁次农历乙酉腊月，琅琊人杨永信忧网瘾之危害，念华夏之将来，尽耿耿丹心，敢举世为先，建临沂网戒中心，创易魄修心模式。三维一体，百余环节，以爱为器，重塑人格。挽冠童于危途，救家庭于水火。历经七载，顺天惠众，积善五千例，弼社会和谐。国彰其功，享政府津贴；民感其德，谓中国好人”。<br/><br/>编者随笔：“爆竹声中一岁除，网戒中心换桃符”。送走了成绩卓越的2012
