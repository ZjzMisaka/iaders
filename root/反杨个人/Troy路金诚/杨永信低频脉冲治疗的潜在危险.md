/><br/><br/>技术角度解析电休克疗法被叫停后网戒中心使用的“低频脉冲”疗法真的安全么？<br/><br/><!--more--><br/><br/>?抛开临沂网戒中心涉嫌违法不谈，本文试图从医疗仪器的角度解析杨永信的低频脉冲疗法为什么会造成那么大的痛苦以及它可能的危害。<br/><br/>2009年7月，卫生部叫停了临沂网戒中心使用的“电休克治疗”，网戒中心并没有因此关门大吉，而是转而使用了新的“低频脉冲治疗法”。网戒中心号称这种“低频脉冲治疗仪”是广泛应用于大中小型医院和家庭康复理疗的产品，能让戒治者放下抵触情绪，更加认真地进行交流。<br/><p class="picbox"><img src="http://ww3.sinaimg.cn/large/c530e421jw1f7kxygtns5j20hs0c3402.jpg" alt="有记者体验“低频脉冲治疗”" /><span class="picinfo">有记者体验“低频脉冲治疗”</span></p><br/>?<br/><br/>作为生物医学工程专业的学生，笔者在医院实习时曾有机会接触过这个?“低频脉冲治疗仪”，它被用在康复科，有的医院甚至有专门的“低频室”，它可以用来缓解疼痛、酸胀。那么为何这样一个缓解疼痛的康复理疗设备在杨永信手里会变成一个制造疼痛的装置呢？<br/><br/>答案从下面媒体采访的一段文字中可见一斑：<br/><br/>“两块铁片贴在太阳穴上，连上电源，第一次被‘脉冲’的彭俊‘虽然有些难受，但是还可以，不是很痛’。第二天，彭俊又被带进治疗室，这次“治疗” 时不仅有铁片贴在太阳穴上，还有两根针插在眉心和人中上，感觉也和昨天大不一样，‘真的受不了，非常痛苦，难受得喘不过气来。’彭俊告诉记者，一次脉冲的 时间在3-8秒，然后停下来，之后又是新一轮的脉冲，如此反复，‘停下来就是让你喘口气。治疗室一直放着氧气瓶，万一喘不过气时可以紧急使用。’私下里， 彭俊曾和经历过‘电击’和‘脉冲’两种疗法的老盟友聊过：‘他们说，贴铁片再扎针，是脉冲强化治疗，比以前的电击还难受。杨永信有次直接跟我们说，脉冲疗法和电击疗法就是换汤不换药。’”<br/><br/>而事实上低频脉冲疗法也是在用电流对人产生刺激，只不过电流要小很多。这样的小电流一般不容易造成疼痛，“盟友”们口中的“杨叔”想了个办法，电流小，那就减小电极的接触面积。<br/><br/>专业上对此类电刺激疗法的安全限定由电流密度，电荷密度以及总电荷量来计量。一般的电刺激类治疗用的平面电极大小为25~35 cm2，而常用的26号~28号电针直径为0.35~0.40 mm，接触面积缩小20000倍，杨永信将本来的平面电极换成电针，输出电流不变的情况下，电荷密度就翻了两万倍！<br/><br/>Liebetanz和他的同事们在论文<i>Safety limits of cathodal transcranial?direct current stimulation in rats</i>中，通过在大鼠上的实验获得的零病变阈值电荷密度大小为52400 C/O，超过这个值就可能造成大鼠病理可见的脑组织损伤。而 MA Nitsche等人则在文章<i>Safety criteria for transcranial direct current?stimulation (tDCS) in humans</i>中表示，在没有得到比目前所掌握的情况更多的数据的前提下，建议针对人的电刺激治疗不应超过他的团队的实验中所采用的标准，即电荷密度220 C/O；哥根廷大学的实验则把这一数值提高到343C960 C/m2，总电荷量为2.4 C。根据网上提供的信息，杨永信所使用的G9805-C低频电子脉冲理疗仪的参数显示，“治疗仪输出幅度最大时，每一个脉冲的电量应大于7 μC”。根据与@未消逝的青春2015沟通中获得的信息，杨永信采用的就是该仪器的最大频率100 Hz，最长的治疗时间为两小时左右，通电的时间占一半。这样电荷密度在杨永信把平面电极换成针电极后达到了5645.16 C/m2，单个仪器总电荷量达到了2.52 C，而根据他所见最多有4台仪器同时使用，听说5台也很常见，这样的话总电荷量甚至达到了十几库伦，两个数据均在针对人的电刺激安全标准的5倍以上。<br/><p class="picbox"><img src="http://ww2.sinaimg.cn/large/c530e421jw1f7l1n91uzmj20hs0bxab9.jpg" alt="杨永信使用的低频脉冲治疗仪" /><span
