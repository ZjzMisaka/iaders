/><br/><!--more--><br/><br/>我们家买电脑算是比较早的，所以我从小就接触到电脑和网络。那时并没有产生什么兴趣，甚至是连用都不会用。如果按这样的情况发展下去，我也就不会走偏。但这并没有成为现实。<br/><br/>在一些同学的“指引”下，我学会了上网，学会了去玩一些小游戏。每个周也只是玩一小会，和现在的学生每天都抱着手机玩相比，可以说是小巫见大巫。游戏这个东西，真的是有一种力量吸引着你。你赢了，有种成就感，就想接着玩下去，你输了，却又激发了你的挑战欲望，不赢不罢休，游戏开发商也就是抓住了这种心理，才让人能玩下去，玩进去。<br/><br/>一周一次的时间满足不了我，我就想有更多时间的去玩游戏。但每天晚上写完作业就没什么时间了，自己就想出一个“好办法”――少写一部分作业。这确实让自己尝到了甜头，于是自己每天写的作业越来越少，因为自己的欲望在不断的膨胀，直到有一次，自己被老师发现作业做的太少而被叫了家长，这件事才败露。从那以后，自己就被父母严格控制着上网的时间，他们不想让我就这样把学业荒废了，因为当时我才是一个二年级的小学生.我也后悔过，也发过誓，以后不再那么没有节制了。<br/><br/>但这没有用，表面上看是没事了，但是那颗贪图享乐的心还在，这只能是一个开始，是一个祸根，而不会是结束。这个祸根像千里之堤上的蚁穴，迟早会让这个大堤毁于一旦，这只是开始的结束。<br/><br/>到了三年级，我又玩上了游戏，是比以前节制多了，也能完成作业了。但现在所玩的游戏和小游戏又不同，它加入了经验与等级制度。小游戏只是单纯的闯关，但是等级制度却一直存在,不是单纯的闯关。不升级，就不会有更多好玩的东西，而等级越是往上越难升，也就出现了有人单纯为了升级而玩。而这时需要大量的时间与精力，我们就一直被这等级与经验所困绕。时间长了想放弃，又不忍心，这也就是游戏开发商把玩家留下的另一种方法，我也又被套在了上面。<br/><br/>四年级，我通过同学的介绍第一次接触了网络游戏，也就是大家都熟悉的那款《穿越……线》，在里面用枪、匕首等真实武器去杀人，是很痛快，感觉能发泄心中的情绪，但更容易让人着迷，让人上瘾。我开始和父母谈条件，父母只允许我在周五玩一晚上，其他的时间不让玩。我哪里会听他们的？在父母不在家的时候，我就偷玩电脑，即使有密码，就算是猜上一上午的密码，我也不去学习，不去写作业，因为我有了一种不好的思想，就是厌学。<br/><br/>和现在的孩子一比，我算是正常的，每天用手机玩游戏，对学习又不上进，但也是在学校里上课学习，这一切都好像很正常，很差不多，就如现在的人们看到“低头族”都见怪不怪了。这真的是正常的吗？我想并不是，而就只到这就停下了吗？我想也不是。<br/><br/>在与父母的关系上，我们并不和以前那样和谐，而是产生了更多的矛盾与争吵。也就让自己更有了逆反的心理。与父母作对，也就是这样形成的。<br/><br/>每天上课心不在焉，回家以后作业应付，对父母态度不好，满脑子都是游戏，这样的状态一直持续到了我上初中之前。也就是这样的状态，我在小学里的成绩还属于中游，说明我并不是学不好，只是没有去学。每天过的这样的逍遥自在，让我觉得这才是青春，这才是人生啊。<br/><br/>现在一想，我错了，但那时我却没认真的去想过。<br/><br/>很自然，小升初的考试，我的成绩并不理想，但父母为了让我有一个好的环境，让我去了最好的初中，可想而知，我是倒数。经过几次的考试，我一直“坚守”着自己的“岗位”，一直是倒数。这让我对学习更没有了信心，我最争气的是我和班里一些成绩一直在上游的学生玩到了一起，但向他们学习的只有如何去玩游戏。<br/><br/>对学习没有信心了，就把更多的精力放到了游戏上。为了能更好的玩游戏，高人一等，我开始买游戏装备。但一个学生的经济来源能有哪些？只不过是从父母那里拿罢了。但父母会这么轻易的就给我吗？不会，于是我就用更加直接的方式――偷！有时十块五块，有时两三百。这样算下来，我也偷了有几千了。<br/><br/>为了游戏，我做过很多事。父母把电脑藏起来，我就找出来偷着玩。被父母发现了，消停一两天，过几天接着玩，我还找出了一个不用的手机，晚上在杯子里面玩，直到深夜。有一次暑假，爸爸发现我不写作业偷偷玩游戏，一气之下把电脑都摔了，但我仍不知悔改，就找出其他的电脑继续玩。<br/><br/>那时早已不是去“上学”了，可以说是“混学”或是说“熬学”了，上课根本不听，作业全靠抄，一点也不想去上学。现在再去评价一下那时，就一个字：傻。<br/><br/>到了初三的第一个中秋节，我在抄同学作业的时候被发现了，爸爸气愤的说了一句：“你干脆不用上学了！”我一听，正合我意！也就这样答应了。<br/><br/>现在拿我的这段过去和现在我的一些同学相比，有很多相似的地方，不知道这些同学又是否会和我一样，越走越偏，但我知道，这样下去不会有好结果的。<br/><br/>于是乎，我不去上学了，家里早就断了网，于是我在家里就天天闲着没事干。为了玩游戏，我去了网吧。说实话，网吧并不好。环境差，主要是烟味很大，人也很杂，说不定什么时候就会出事。爸妈也知道了我去网吧的这件事，为了让我能留在家里，他们也只好通了网。<br/><br/>这样我就更肆无忌惮了，对爸妈一句话不说，就像陌生人一样，每天只是在电脑前玩游戏。睡眠的时间越来越晚，起床的时间也越来越晚，以至于最后变成了早上六点睡，中午一点起的作息习惯。每天这样的生活，让我不再像是一个人，而是一个为了游戏而生活的机器。现在回想那段时间的我，自私自利，不考虑他人，更不考虑现实。<br/><br/>我没有去考虑过现实，也没有考虑过未来，但是我的父母帮我想了，因为他们爱我，不想我以后只是一个社会上的寄生虫，所以他们四处打听，四处奔波，来为我找一个能帮助我回归现实的地方。天下哪一个父母不爱自己的孩子？只是爱的方式不同。一味的姑息、纵容孩子，那只能叫溺爱。而对孩子的严格要求，也是一种爱，更是一种责任。<br/><br/>幸运的是，我的父母找到了临沂网戒中心，并下定决心把我带来，这也是我一生命运的转折。但是仍然有很多的孩子像我一样沉迷网络游戏，可他们的父母没有找到中心，甚至有的父母直接放弃了改变孩子，任由他们自生自灭。<br/><br/>来到中心的我是幸运的，因为我得到了改变的机会。每天的心理点评课，是我改变的最重要的环节。<br/><br/>每天的点评不仅仅是对别人的点评，也是对我们自己的点评。正像那句“以人为镜，可以知得失”一样。每天的日记也是对自己的一种反省。我们的改变就是在这每天的环节中发生的。我们内心原来一些错误的观念和不良的习惯，都慢慢的发生变化。拥有一颗感恩的心，一颗孝心和一颗明辨是非的内心。这个过程并不是一两个小时就能发生的，而是需要一个漫长的过程。但是与“江山易改，本性难移”这句话相比，相对来说比较漫长的时间与根本无法改变不是更好吗？<br/><br/>就这样，我在这里发生了改变。这种能让人内心发生改变的地方，我不能肯定只有中心这一个地方，但是我所知道的只有这个地方。不仅是如何做人，我还学会了学习，只有这样，在学会做人之后，学习才会变得有用。古今中外有多少人很有文化，但是没有做正常人所做的事，而干了一些危害他人的事。中心教会我要先做人后做事。<br/><br/>其实知道了如何做人，就明白了学习是为了什么，为了谁，明白了这些，在学习上自然就会自己推着自己去学习。在中心的学习组里，就是给你自己去摸索，去寻找最适合自己的学习方法的。这同样也是一种改变。<br/><br/>十一个月，我经历了很多，学到了很多，改变了很多。当我得知我要离院的那一刻，我很不舍。十一个月，和大家相处了那么长时间，会有感情，突然要分别当然会不舍。但天下没有不散的宴席，离院才是改变的真正开始。那一天，很激动，内心就像是五味瓶，什么滋味都有。当我走出中心大门的那一刻，我哭了，但这没办法。<br/><br/>出院后没几天，我又回到了学校，而我也只能从初二的下半学期开始，因为基础太差，从头开始都不一定能跟得上。和原来的我不同，我知道上课要干什么，回家要干什么，作业要怎么做，这些我都很清楚，因为中心教会了我如何去做。和父母也不再是陌生人，因为我找回了亲情，也懂得孝顺父母。我变得和以前不再一样，我现在完全是另一个我。<br/><br/>第一次月考成绩出来后让我有些无法相信，我考了年级第四十名，其中数学是单科最高分！我很激动，也很感慨，这是我自己的努力，也是中心的功劳。没有中心我定不会有这样的成绩。<br/><br/>因为接受了中心的军训，所以在上体育课和出操时都比较的认真，结果我被体育老师挑选为体育委员，带领着全班赢得了运动会和会操比赛的第一名。如果不是中心让我锻炼自己，我也不会有现在的领导能力。学校的演讲比赛我也参加了，获得了第二名，如果不是中心的大小演讲和大小辩论赛，我的口才也不会那么好，有可能我连上台都不敢。<br/><br/>现在想一想我没有来中心会是怎样，也许我还在电脑前玩着游戏，浪费着大好的青春年华，亦或是我早已猝死在电脑前。而现在的我却在这大好的青春年华里做着我应该做的事情。<br/><br/>暑假里我又回到中心深造，很大的原因是因为中心的动力挖潜组。这也是一个让我受益匪浅的假期。表面上看只是简单的背诵单词，但这是一个让自己摸索方法，不断让自己突破的过程，最主要的还是心态，用一颗平常心去对待每一次的成功与失败。虽然我们每天的生活节奏很快，但是我们很充实很快乐。一个多月，我学完了初中所有的单词和短语，这比任何一个补习班来的都要快，效率更高，效果也更好。<br/><br/>回到学校，面临紧张的初三，我不怕，更不用说我最薄弱的英语得到了强化，所以我更有信心。<br/><br/>我的青春还没有结束，还有更多的东西去让我挑战，我不怕，因为我是中心走出来的孩子。<br/><br/>我的改变让我想了很多。我为什么会走偏？原因有很多很多，我就说自己想到的几个方面：<br/><br/>1.人天性上的一些特性。大家都看过《格列佛游记》，里面的“慧S国”的那一段，我想很有代表性。和人类长相差不多的耶胡，让主人公起初很是讨厌，因为耶胡的性格贪婪自私，而人类是有理性的。随着与慧S的接触，主人公发现人类就是有理性的耶胡，而人类就是凭借这一点理性，做的都是一些为本性上的自私、贪婪、享乐而服务的事。归根到底还是人本性上的问题。人身上都有自私，只是有的人可以克制住，有的人克制不住，因为他们想不明白自私的后果和代价。放眼古今中外，有多少人是因为自私而毁了自己？不断膨胀的欲望，严重的逆反心理，都是使我们走偏的帮凶。<br/><br/>2.家长的教育。孩子的走偏和家长是脱不了干系的。家长对孩子的溺爱包办，会让孩子养成唯我独尊，自以为是的性格。迁就纵容会使孩子没有规则意识，对父母没有孝心。而过于严厉的批评，没有一点的表扬和鼓励，也会使孩子产生不自信、自卑或是逆反的性格。但有一点要注意，有的家长的教育方式一样，但教出来的孩子不一样，这也是说明了根本的问题还是在孩子身上。<br/><br/>3.外界的影响。现在社会上消极因素很多，特别是网络上，有很多的垃圾信息。学校里也不是绝对纯净的地方，因为那里也有一些“混学”的孩子。面对这些消极因素，如果自控力好，没有什么太大的影响，但如果从众心理太强，也许就会和他们同流合污。<br/><br/>以上就是我认为的比较重要的几点，其实还有很多的原因，但最主要的还是自身的性格缺陷。<br/><br/>最近看到网上有很多人说中心不好，我很奇怪，为什么要这样？真正的中心是什么样，我也想谈一谈我的看法。<br/><br/>一、心理点评课堂<br/><br/>这是我们改变的重要环节，每天的问题都是在这里解决。也许有的盟友是几个月都点评不到一次，有的却是天天被点评，这就是通过别人的问题来看自己的问题。大多数人问题的核心就是自私，所以可以通过一个人就代表很多人。情感课是最能帮助我们找回亲情的环节，而点评师有时的一句话就能改变我们对一件事，或几件事的看法，从而改变我们的内心。有人说这是在洗脑，不错，这是在帮我洗掉消极的，然后教会我积极的。这种“洗脑”不好么？<br/><br/>二、氛围<br/><br/>这里的氛围比在其他任何地方都好。因为这里是一个大家庭，每一个人都相互的去帮助，不仅是生活上帮助，在改变上也是相互帮助，相互指出问题，一起改正，也是一种帮助。一个小室里的盟友都很团结，每天的演讲和打扫卫生都可以锻炼我们。这里是一个互帮互助的大团体，才会有良好的氛围，这也是为什么我们把这里当成自己的家。<br/><br/>以上就是我对临沂网戒中心的一些看法。<br/>首次转载时间:
