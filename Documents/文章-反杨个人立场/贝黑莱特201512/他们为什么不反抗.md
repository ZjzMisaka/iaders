<br/><img src="http://ww2.sinaimg.cn/crop.0.0.1919.1079.1000/006wDpsYgw1fa20qcxw4xj31hc0u0n45.jpg" /><br/><br/>所以，他们为什么不逃跑？为什么不反抗呢？<br/><br/><!--more--><br/><br/><i>向我们提出的所有问题中，从来不缺少一个问题；事实上，随着岁月更替，这个问题出现地越来越频繁，并且带有越来越浓厚的谴责意味。与其说它是一个问题，不如说它是一系列问题：你们为什么不逃跑？为什么不反抗？为什么不“事先”避免被捕？我们无法回避这些问题。并且，随着时间的消逝，提出这些问题的次数越来越多。所以这些问题值得我们重视。</i><br/><br/>――普利莫・莱维《被淹没和被拯救的》<br/><br/>在#临沂网戒中心#和#s母少女与网戒学校#的相关讨论中，时常见到有这样的提问：为什么没有学生反抗呢？为什么不团结起来？<br/><p class="picbox"><img src="http://ww4.sinaimg.cn/large/006wDpsYgw1fa202n4vejj30fm07y0v5.jpg" alt="澎湃问吧的提问" /></p><br/><br/><h6 class="picbox"><span class="picinfo" style="color: #999999;">澎湃问吧的提问</span></h6><br/>?<br/><br/>还有人问过类似的问题：你们为什么不“事先”防止被抓捕呢？这个问题在网友“未消逝的青春2015”身上或许可以得到一定程度的解答。他因为在微博上发文揭露临沂网戒中心的事情，被“家委”威胁，在察觉到危险时，逃离了临沂。然而这个事先“逃离”的经历难以复制，而且，这是有过前一次被送进网戒中心的经验，而前一次不存在“事先避免”。<br/><br/>事先避免被捕或许还有些用，被抓住后，那就只能老老实实待着了。<br/><br/>我也曾想过，自己若被送进类似的学校和机构，一定不惜一切要逃出去，或者干脆和加害者同归于尽；又或者团结一些人进行反抗；或者，不妥协，不合作，进行“弱者的反抗”；实在不行，还可以自杀。因为我也听过一句话：哪里有压迫，哪里就有反抗。<br/><br/>“有压迫就有反抗”在网瘾学校里真的适用吗？答案很明显：不适用。<br/><blockquote>我愿意邀请任何敢于亲身作出判断的人，诚心诚意地，进行一个概念化的实验：让他想象（如果他可以），长年累月生活在一个犹太人隔离区，遭受着慢性饥饿、疲劳、混乱和羞辱的折磨；他已经见证到身边的死亡，他所爱的人，一个接一个死去；他与世界隔离，无法接受或发出信息；终于有一天，他被塞进火车，80到100人挤在一个闷罐车厢里；在无眠的日日夜夜，他被押送到未知的目的地；而最终，他被投进高墙中的一个无法理解的人间地狱。</blockquote><br/>――《被淹没与被拯救的》?<br/><br/>这里进行一个类似的想象：有一天，你被家人和亲戚外加几个不认识的人抓住，塞进一辆车；在经过漫长的几个小时后，被押送到未知的目的地；在这里，你遭受着虐待、疲劳、混乱和相互检举的折磨；你见证身边人所受的折磨，同时遭受着身边人的折磨；你日日夜夜生活在这里，你与世隔绝，无法接受或发出信息；你被投进高墙中的一个无法理解的人间地狱。<br/><br/>对这些学校和机构里的学员来说，逃跑是困难且极为危险的。在临沂网戒中心的相关报道中，逃跑失败后受到的惩罚是相当恐怖的，“杨叔专场”恐怕是每个盟友都深深恐惧着的刑罚。<br/><br/>在一个强制要求盟友互相监督、检举的地方，成功逃跑意味着什么呢？因为逃跑难以实施，那么，逃跑者不可能没有同谋，每个人都注视着彼此，不可能没人注意到。所以，无计划的逃跑是徒劳的，除非你遇到极好的运气。<b>到目前为止，我并没有见过一则在临沂网戒中心里逃跑成功的案例</b>。<br/><br/>在这些学校和机构中，都要求学员必须穿迷彩服，进行所谓的“军事化管理”，准确说应该是“准军事化管理”。在军训时，也是要求穿迷彩服的，在所有人都穿迷彩服时，大家都差不多，留着差不多的发型，做着相同的事情。在网瘾学校中，大家也都穿着同样的迷彩服，做着同样的事情，而<b>这在一定程度上阻止了逃跑的发生</b>。迷彩服是一个标志，和囚服一样，可以表明身份，虽然看上去区别不大，但人们一眼就能认出他们的迷彩服。而机构中的虐待，恐怕早已摧毁了他们的意志。<br/><br/>就算是本地人，被带到这样的机构，在信息极度匮乏又没有人帮助的情况下，也是难以掌握机构周边的情况的。对于外地人，不知道自己具体在哪，附近也没有熟人，<b>寻找到切实可行的隐藏地点是极其困难的一件事</b>。<br/><br/>对于临沂网戒中心，因为是矫正机构，又是四院，也就是精神病院，在其中逃出来的人都算是精神病患者，至少在一些人眼里是这样的。<b>而接精神病人出院，按一般流程，需要“谁送谁接”，由父母送就由父母接，也就是“家务事”，这大大增加了逃跑后受到援助的难度</b>。<b>而逃跑后的结果――逃跑者受到严厉的惩罚，而其他哪怕无关的人，也受到牵连，也就是所谓的“全员责任”。</b><br/><br/>看一个例子：<br/><br/>在《我在临沂网戒中心的真实经历》中有这么一段描述：<br/><blockquote>后来又来一个女生，可能是她本身就有病，然后要被带着出去看病，在看病的时候她提出去厕所，因为是女生，男家长也不方便进去看着，结果那个厕所有两个门，女生就跑了，过了一会女家长进来一看没人，就到处找，找到的时候已经跑到马路上了。家长就在后面追，女孩跑过马路，看到对面有个治安岗亭就跑过去拍玻璃喊救命，结果还是被家长抓住了。治安岗亭的问了问，<b><span style="color: #d24a35;">家长就说她孩子精神有问题</span>，出示了身份证之后，值班的也没多管闲事，抓回去的结果可想而知</b>。一场杨叔专场治疗的一星期不能跑操，只能由专人看着站在边上，除了那个女生受罚之外，他的家长也被罚款3000，除此之外所有的家长都被罚款，称为<b>全员责任</b>，理由是：如果这里关门了，谁来拯救你们的孩子！所有人都罚款以示惩戒，不过数目自己看着来，有的就当场表态交1000，有的800，有的600，有人带头别人也不好意思不交，纷纷表态。然后我妈交了20。</blockquote><br/>那么自杀行不行呢？我见过从临沂网戒中心出去后自杀的案例，也听过以身体不适为由，去医院后自杀的，而在临沂网戒中心当中自杀成功的报道，我还没见到。倒是有自杀失败的案例：?<br/><blockquote>点评过后小卫要求去厕所，在从厕所出来的时候，他趁随同的家长不注意，一下子冲进一间小室，拿起水杯就砸玻璃，还用脚去踹玻璃，想打碎玻璃跳楼，<b>可惜，那些玻璃都是钢化的，玻璃没有碎，他也没跳成，反而被冲进来的家长和盟友抓住了</b>。然后被押回点评课堂。。。。。</blockquote><br/>和逃跑失败后一样，小卫受到了非常恐怖的惩罚：<br/><blockquote>所有准备工作之后杨永信什么也不说，直接打开两台仪器。我也不知道是哪两台仪器，反正小卫躺在床上不停的抽搐，太阳穴的针旁则是留下一行鲜血，第一次的电击估计超过五分钟，然后杨叔开始和他说话，无非就是你做错了，你让父母担惊受怕了，你对不起你的父母之类的话。在被电击后，人是最无助的时候，那时候说的话会在你的记忆里留下深深的烙印，比如我，事情都过去一年多，我现在还可以清楚的记得我在那里面做了几次的治疗，每次的治疗过程是什么，我都记得一清二楚。杨永信不断的训斥小卫，小卫则是不断的哭着求饶认错。而杨永信则是冷笑“知道你还这么做，你对的起谁！”然后打开了所有的仪器。这一次持续了十几分钟。而杨永信则是不断的转动四个仪器的螺旋按钮，有时会咯嘣的一声脆响，然后又是一声脆响，我知道那是仪器在关闭然后马上再打开，造成那种一波又一波，让人无法忍耐和适应的疼痛......</blockquote><br/><blockquote><b>小卫的感受我不知道，但是想必比死了也难受，不然他宁愿跳楼都不想承受这种痛苦</b>。这场治疗持续了两个多小时，到了后来杨永信也累了，就坐在椅子上，打开仪器不断的转动电流旋钮，还叫个安全小组帮他捶背，捏肩膀。到了后面小卫躺在上面就像是一个死人，双眼空洞无神，嘴角不断留出口水，只有打开仪器的时候身体在不断的悸动，反抗的动作都没有了。对杨永信的话只会说，是，我错了，杨叔。</blockquote><br/>说完为什么不逃跑，说说为什么不反抗吧。其实逃跑和自杀也算是反抗的一种形式，这里说说明面上的反抗吧，像是奴隶起义那种。<br/><br/>“有压迫就有反抗”，要激起反抗，压迫是必须存在的。教科书上所描述的农民起义大多是“官逼民反”――由于掌权者压迫底层“被压迫者”，激起愤怒和民怨，所以农民奋起反抗。但是，教科书上的起义通常都是王朝末期，盛世中的起义和反抗却很少书写，这些反抗多数都失败了，甚至在历史上没有留下痕迹。<br/><br/>具体回答“为什么不反抗”很难，不光是这个问题涉及面很广，需要从心理学等专业层面解释，还有个原因，那就是<b>我们难以理解甚至无法理解他人的经历</b>。你顶撞一次家长或者老师，也许只是被骂一顿，而在网戒中心，可能就是一次专场，其他机构，也许就是一顿暴打，甚至可能丢掉性命。<br/><br/>说这些机构中的人没有反抗并不准确，因为没有进去过的经验的人，通常都会在刚进去的时候激烈反抗。<br/><br/>电视节目 《说事拉理》再探“电击”网瘾少年 里面，<b>一个27岁的青年，被家人灌安眠药，绑住手脚，送到杨永信那里，在镜头前，他也激烈反抗，还起身打过一个盟友，但很快被制服</b>。之后的情况不得而知，节目没有继续报道。而在其他节目中出现过的一个尖叫着被抬进治疗室的那位少女，在《说事拉理》这期节目中，变得非常温顺。其他节目中也描述过，进来会打人、打窗户准备逃出去的男孩，又或者扇家长耳光的女孩，在接受“治疗”后，见到父母立马下跪，前后完全判若两人。这在一定程度上说明了反抗的难度――<b>“治疗”能够损害人的精神力量，这种压迫非常高效</b>。<br/><br/>我不清楚网戒中心里是否有过其他的反抗，不过可以肯定，<b>在这里面反抗是毫无意义的</b>。其他机构有体魄强健的教官，像是电影里的狱警一样，一两个狱警就能管住一个监狱，除非发生影视剧中的监狱暴动，否则充分的武力足够在短时间内镇压手无寸铁的反抗者，更何况这里面关着的多数还是青少年甚至儿童。在网戒中心，有着“家委”这样的组织，家长作为保安和狱警，只要有一两百人在网戒中心进行治疗，那么这些“保安”的数量绝对多于任何一个监狱，里三层外三层的包围，反抗如何进行？<br/><p class="picbox"><img src="http://ww2.sinaimg.cn/large/006wDpsYgw1fa20joxphlj30ci0gowft.jpg" alt="家长手拉手监视盟友" /></p><br/><br/><h6 class="picbox"><span class="picinfo" style="color: #999999;">家长手拉手监视盟友</span></h6><br/>?<br/><br/>那么为什么不团结起来呢？哪怕是网戒中心，所有盟友团结起来，人数上足够与家委抗衡，而其他机构，教官再强壮，咱十个打一个还打不过？<br/><br/><b>团结需要一个可以交流的环境</b>，需要你向其他人发出讯息，其他人可以发出反馈，然后大家商量团结起来做什么事，除非所有人或者至少是大多数人拥有极佳的默契，可以通过教官和家委察觉不到的暗号，在同一时间进行反抗。<br/><br/>但是在所有的类似机构的报道中，都提到里面不允许相互的交流，空间的限制使得成员间难以交流，成员之间无法达成信任，更无法密谋什么。在网戒中心，86条“可能‘点现钱’行为”里面明确禁止成员的交流――“11.传纸条”、“31.严重警告之后盟友之间仍出现小团体现象”、“55.与盟友之间出现交往过密嫌疑”等都明确限制了成员的交流。86条中还有其他的要求，使得使用非口头的交流方式也难以实施。<br/><br/>不相互交流，那么靠默契呢？每个成员进入机构的时间不同，时间、空间、年龄、观念等也无交集，最信任的人估计只有父母，而正是父母送他们进的机构，他们正对加害者感恩戴德呢。与原先可能最亲近的人无法建立信任，成员间如何建立信任？难上加难。其实也不是难上加难，网戒中心的家长们就很团结，但他们能够交流，相互“诉苦”，并且有个团结的中心――杨叔。而是对于其他成员而言，在经历了老成员欺负新成员、成员间相互检举后，又无法有效交流，信任几乎无法建立。<br/><br/><b>因为交流障碍和相互不信任，所有的反抗都可能被扼杀在摇篮里，而反抗的典型将会受到相当严厉的惩罚，并以此警告其他成员。</b><br/><br/><b>其实不光是交流障碍和相互不信任，很可能成员一直处于精疲力竭的状况</b>。有过许多类似机构死人的案例，在相关报道中，体罚是常有的事情。机构有军训、跑步或者其他运动形式来“锻炼”人的身体和意志，然而实际上，这种锻炼往往只是造成身体的疲劳，使得你长期处于疲惫的状态。以网戒中心为例，这里有一个磕操的仪式，用于净化心灵，而这套仪式做下来，疲惫时必然的事情，而除了疲惫，还可能会伴有腰椎和膝盖的组织损伤。也就是说，成员每天处于身体和精神双重疲惫的状态下，浑身酸痛，或伴有身体损伤，就算能够相互交流，也难以进行反抗。<br/><br/>那么到底有没有过起义这种反抗呢?我不知道，我知道的只是，如果有，他们的结局一定很惨。到目前为止，除发生影响恶劣的事件的相关报道外，几乎都是从类似机构出来的人进行披露，有的在贴吧、微博、微信等社交平台上，也有接触到新闻机构的，他们说出自己所掌握的事实。然而，就算引起巨大的关注，这些机构仍然在运行，仍然有许多人不愿意倾听或者相信他们的话，因为总是充斥着撒谎者、混淆视听者（比如贴吧，难以确定真实性的事情很多）以及觉得“虽然不合法，但社会正能量增加了”的人。<br/><br/>引用《被淹没和被拯救的》中的一段：<br/><br/>正如马克思知道的，在真实的世界中，革命并不是由这些人完成的。只有在文化和电影的浮华辞藻中，才会出现这样的革命。所有的革命，改变了世界历史进程的革命，以及我们在这里讨论的微不足道的革命，都是由那些非常了解压迫却并非切身之痛的人所领导。<br/><br/>所以，他们为什么不逃跑？为什么不反抗呢？<br/><br/>相关文章：<br/><br/><a
