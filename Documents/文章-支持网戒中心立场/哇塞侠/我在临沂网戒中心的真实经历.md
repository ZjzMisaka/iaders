  “现在你的父母，舍弃了自己的工作，来到这里和你一起改变，你能安心地在这里住院吗？”<br/><br/>我点点头。<br/><br/>走到护士站，看到爸妈神色慌张地站在那里，眼神中包含着许多感情，但无一例外都充满着爱。我过去抱住爸爸妈妈，哭了，这样的温情，我已很久没有体会到了。孤独、无助的日子我已经过了太长太长的时间了。<br/><br/>不过，感动只是一刹那间，人的习惯性思维不可能一瞬间扭转，和大多数盟友都一样，很多时候我只是口服心不服。但在慢慢适应的过程中，发现一切都与自己所构思的“魔鬼”截然不同。<br/>?<br/><br/>记得第一次见到返院盟友的分享，他们每个人取得的优异成绩，都给我带来了激励和鼓舞，让原本对生活失去希望，准备浑浑噩噩混下去的我再次燃起了信心。坦诚而言，我是一个心气很高的人，也没有人不想自己有一个好的未来，但眼高手低，“思想上的巨人，行动上的矮子”，让我们所得到的往往达不到预期的目标。每一位精品盟友成功的背后，是他们不懈的努力与付出，是他们经历过比别人更多的挫折和坎坷。为什么他们能如此努力，如此付出？他们常挂在嘴边的一句话是：“有中心，有杨叔，才有现在的我！”<br/><br/>记得第一次上情感课，在点评师的悉心引导下，我的心门渐渐打开，那颗冰冻已久的心在渐渐融化。聆听着一句句饱含感情的话语，眼泪止不住的往下流淌。曾经的一幕幕不断地在我脑海中浮现――呀呀学语时，蹒跚学步时，总有一双熟悉的身影在身旁指导我，小心翼翼的怕我受伤；上学的第一天，是父母手拉手把我送入了校园，送进了学校的大门；在学校和同学发生矛盾，或是被人欺负，是父母不断引导自己，帮自己解决心中的苦恼。还有太多太多，这些成长中被忽略的点点滴滴不断被想起，埋藏在内心深处的美好回忆不断被唤醒。当点评师说：“请走到父母身边”时，我迫不及待的跑到他们面前，用力地紧紧地抱住他们，心中有太多太多想说的话，想道的歉，但在这一刻都化作了满面的泪水。那一刻，我开始庆幸自己来到了中心，让自己本就冰冷的心，再次有了感知爱的能力。<br/><br/>第一次参加周末活动，简单的“人椅”、“纸杯传水”等，都为我带来了轻松和愉悦。盟友间的通力合作，共同付出；小组间的激烈竞争，互不相让，让中心的生活变得多姿多彩。忘了有多久没有开怀大笑了，忘了有多久没有这样轻松愉悦了。虚拟的世界所给我带来的欢乐是短暂的，一场游戏的胜利所带来的喜悦感也仅仅是几分钟而已，关上电脑后，空虚的反噬更加强烈。“快乐”、“幸福”、“愉悦”、“轻松”，这些字眼已经在我的字典中寻找不到了。本该属于这个年龄阶段的阳光、快乐，在这一刻，在失去很久以后又重新回到了我的身上。<br/><br/>第一次辩论赛，第一次演讲比赛，第一次……这么多的第一次，让我的生活再次变得充实。<br/><br/>在中心，有一首歌深深打动着每一位盟友的心，每每听到那熟悉的旋律，每每听到那朴实无华的歌词，内心就会变得无比激动。<br/><br/>“当我有一天，走进了虚拟的世界，我就再也找不到前进的方向。当别人给我带来关爱的时候，我却把他当作了一世的仇人。在网游的茫茫雾海中，迷失了前进的方向，我没有了生命的航标。亲爱的朋友啊，从网梦中醒来吧！让我们用新的生命，乘着爱的翅膀起飞，追求新的希望，追求爱的梦想。追求新的希望，追求爱的梦想！”<br/><br/>就是这首歌，唱出了我们每个盟友的心声，唱出了我们心底里最真挚的感情，经常在唱歌的过程中，泪水不知不觉中便打湿了眼眶，没有经历过走偏的人，是不会明白的，看似活得随心所欲无拘无束的我们，心里无时无刻不在忍受煎熬。我们很痛苦，我们很迷茫，像黑夜中迷失方向的旅者，找不到一丝光亮，又像陷入网中的鸟儿，拼命挣扎却越缠越紧。我们在痛苦的深渊中咆哮，低吼，又没有足够的能力跳出这个怪圈，这时，我更加庆幸来到了中心，帮助自己突破囹圄，就如同佛家所说的消除孽障一样，灵魂得到净化的同时，我也已涅重生。当唱到“追求新的希望，追求爱的梦想”时，心里充满了对美好生活的向往，充满了新生的希望，那种难以用语言形容的感觉，和春天阳光沐浴一般，无比温暖。<br/><br/>很荣幸，我曾在中心担任过班委，从楼层纪体到总学思兼二楼楼层学思再到二楼楼长，所承担的工作之重要，是我都没有想象到的，俗话说“干一行，爱一行”，每一项工作的内容都有所不同，但我真的非常热爱它们。可能有的人会认为成为一名班委，既有权又有势，绝对会混得如鱼得水，那你就大错特错了。刚刚接任楼长时，整个人都感觉压力山大，因为无论楼层里出现什么样的情况，都有自己的一份责任：安全、氛围，这些无形的、看不见摸不着的东西，需要靠自己一点一点去摸索。每天，面对汇总的盟友的问题，总要苦思冥想许久：怎样解开他们的心结？怎样处理他们之间的矛盾？怎样让大家能进步的更快，变得更优秀。找盟友谈心，向家长了解情况，寻求点评师的帮助……一切的一切，只为了一个目标――大家一起更加快速的改变。当然，人无完人，金无足赤，在此期间也难免会出现很多问题，犯下很多错误，但每一次错误，都让我变得更加成熟。从遇到问题就逃避，到勇于直面挫折，在很多人看来是很难做到的，但在中心，在每一个盟友的身上都在发生着这样神奇的转变，这些经历，在为我们铺平人生道路的同时，也是一生的宝贵财富。<br/><br/>进入学习组，再一次捧起书本，走偏时的那种厌恶和反感已荡然无存。取而代之的是一种溢于言表的激动，抚摸着光滑的封面，轻声道：“久违了，老朋友。”再次坠入知识的海洋，如饥似渴的吸取着营养，打个不太恰当的比喻，就如掉入米缸里的老鼠一样。而经过动力挖潜组750单词＋1050短语＋30句子的洗礼后，如同洗去白玉上的斑驳一样，变得更加“清心”。自从两年前的那一漆黑的夜，我彻底抛弃学习以后，就再也没有想过自己还会有重新拾起书本的这一天。这一切，没有中心是绝对不可能实现的！<br/><br/><br/>九月里的一天，一个让我永远铭记的日子，我离院了，更是我的第二个”生日”！在课堂中央，如松战立，左手捧着离院宣誓，右手紧攥成拳，随着“起誓”口令一声令下，斩钉截铁的举到太阳穴处，拳眼朝下，抬起头，深吸一口气――“今天我将要离院，我不必承诺永不再偏，我的心中没有再偏两个字，我心中有的只有做的更好！更优秀，更强大！我不再做欲望的奴隶，我不再做命运的俘虏，我要华丽转身，扼住命运的咽喉，坚信有一天，我能证明自己……”<br/><br/>其实在全力以赴地喊到一半时，早已泪流满面，在中心九个月的时间，说长不长，说短不短，但却承载了太多太多的记忆，甚至于这几个月所成长的远远超过过去的十七年，一幕幕画面在我眼前浮现，酸甜苦辣，在这一刻都糅杂在一起，变为迎接新生活的动力，注入我的内心。在接过离院保证书的那一瞬间，我才真真切切的感受到中心，就是我的家，不带任何引号！<br/><br/>如今离院已有一个多月了，心中对家的那份想念丝毫没有减弱。<br/><br/>现在的我，虽然上学时间不长，但在很多活动中都能看到我的身影，在班级里我担任体育委员，同学之间关系和睦，老师也对我从原本怀疑的态度到赞赏有加.......一切的一切，都是因为中心的教导，都是因为自己曾去过中心，曾是中心的一位盟友。<br/><br/>现在的我，可以很自豪的说：“我是中心走出来的盟友，是杨叔教出来的学生！”<br/><br/>而就是这样神奇的地方，这样充满爱和信的地方，却在最近一段时间被一些居心叵测之人恶意攻击。<br/><br/>面对那些故意攻击中心，为了一己私利而不顾他人幸福的人，我无话可说，也自知多说无益。但是，我想对那些没有真正了解这场舆论的人，说几句推心置腹的话――没有网戒中心，就没有现在的我！监狱和地狱将是我最终的归宿。<br/><br/>在你们身边，肯定会有这样一个群体：他们自暴自弃，游手好闲，消极颓废，每当你看到他们时都会为之痛心和惋惜。以前的我们，就是如此，但我们幸运的来到了中心，而如果中心没有了，现在的他们，又该何去何从？<br/><br/>“低头族”等新词在不断涌出，游戏占用人们的时间在不知不觉中急剧上增，现在青少年的责任心、爱国精神相比于对网络的热爱，显得那么渺小。如果现在发生战争，又有几人不当逃兵？如果国家和政府，在网络问题上，仍没有足够的重视，那“和平演变”将会变成现实，国将不国！<br/><br/>庄子曾经说过，真正的道理说出来后，让普通人听了是会发笑的。中心现在的模式和理念，因为它独特的超前性，也不被大众所理解。但我相信，在历史滚滚长河中，真理总会有拨云见日的那一天！<br/><br/><br/>盟友小成<br/><br/><br/>首次转载时间:
