<br/>
<br/>临沂网戒中心中尽管没有教官等人物存在, 但对盟友的掌控程度是非常变态的, 网戒中心的规章制度复杂而面面俱到, 惩罚措施环环相扣. 例如有开门就进程序等一干 "程序", 是一种长周期的惩罚, 未得到杨永信的允许无法停止. 因此写一篇文章, 详细的写一下网戒中心对受害者的控制手段. (下文按照网戒中心的习惯, 将受害者称为盟友. ).<!--more-->
<br/>
<br/>1. 入院时
<br/>
<br/>对于新入院的盟友, 杨永信就有很多办法对他们进行心理与生理上的双重震慑.
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;1.1. 断绝一切外界信息
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;进入网戒中心的盟友马上会被搜身,上交一切私人物品, 特别是各类电子产品, 通信工具在今后的生活中更是严禁触碰的物品之一. 断绝与外界的联系, 可以防止信息外泄, 更可以使盟友处在孤立的环境中, 最终只能选择放弃抵抗接受治疗.
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;1.2. 接待
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;网戒中心在入院的老盟友中挑选出一部分人称为 "接待" . 由他们出面用强制性的手段把新来的盟友送进13号室进行醒脑治疗, 也就是电休克治疗, 并负责在治疗过程中用力压住盟友的身体, 最终迫使新盟友留下来.
<br/>&nbsp; &nbsp; &nbsp;据称, 目前由强力壮的家委成员替代接待小组进行起飞 (详见3.1.8) 接待等功能.&nbsp;
<br/>&nbsp;&nbsp;&nbsp;&nbsp;一般新来的盟友要进行20分钟左右的治疗, 很多人只一下就受不了了, 表示愿意留下来. 但是求饶基本没效果,
<br/>&nbsp;&nbsp;&nbsp;&nbsp;医生还是会给你做下去. 家长大多数是被告知盟友是脑子有问题, 必须经过治疗才能清醒, 因此都不会干涉治疗.
<br/>
<br/>2. 入院前期
<br/>
<br/>网戒中心中的生活本已没有隐私可言, 但是入院前期的盟友受到的监控更为严苛, 盟友的任何举动都会在严密的监控之下.
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;2.1. 结对子
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;入院半个月内须由一名指定老盟友陪同行动, 两人在小室外间隔不能超过一米, 哪怕上厕所也必须陪同监视, 擅自脱离对子的盟友会被 "治疗" .
<br/>通过贴身监控, 打击新盟友出逃, 自杀的潜在可能.
<br/>
<br/>3. 院内生活
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;3.1. 电击
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;电击, 又称点现钱, 导入治疗等等, 是临沂网戒中心中最为著名的项目, 一切其他方式所不能解决的问题, 都能用一次电击解决. 如果一次不行, 那还可以有 "杨叔专场" , "开门就进" 等一系列处罚跟进. 一旦决定电击, 名字就会被记入心理导入本 (盟友称之为电击本).
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;网戒中心有很多严厉的规定, 称之为 "高压线" , 是一旦违反 (即触电) 就足以直接 "治疗" 的严重的违规行为. 例如, 自杀, 逃跑, 交往过密（或者有相关嫌疑）, 还包括独自上厕所, 对家长发脾气, 带对子 (详见2.1) 时对子脱节等, 偷偷藏药, 拒绝打针. 实际上只要出于本人 "家长建议" , 盟友都可以直接被 "治疗" , 例如, 有盟友因为挑食被 "治疗".

<br/>&nbsp;&nbsp;&nbsp;&nbsp;除此之外, 不遵守治疗室纪律也是会被点现钱的.
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;治疗室的二十条规定:
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;1. 在治疗室不站军姿的+5纪律
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;2. 在治疗室做小动作的+5纪律
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;3. 在治疗室内嬉笑的+5心态
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;4. 靠在墙上的+5纪律
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;5. 交头接耳的+5纪律
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;6. 当杨叔问话的时候说话声音小的或不说话的+5心态
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;7. 随便打报告的+5纪律
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;8. 在治疗室内吃东西的按情况+5纪律
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;9. 不服从管理的盟友+5
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;10. 没有经过允许乱动的+5纪律
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;11. 和杨叔或者班长狡辩的+5心态
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;12. 治疗室说话不卡拍的(和杨叔说话不一致)+5纪律
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;13. 当别人做治疗的时候伸头看的+5心态
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;14. 治疗室面部表情过多的+5纪律
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;15. 杨叔没有说做完就鼓掌的+5纪律
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;16. 没有经过允许私自压人的+5心态
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;17. 没经杨叔同意私自开门者+5纪律
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;18. 私自碰治疗仪为触电行为
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;19. 有意扰乱治疗室纪律的视为点现钱
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;20. 多次违反纪律者直接点现钱
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;2009年之前, 网戒中心使用的是DX-ⅡA型电休克治疗仪, 7月13日, 卫生部在其网站上发布 "关于停止电刺激(或电休克)治疗 "网瘾" 技术临床应用的通知" , 认为, 电刺激(或电休克)治疗网瘾技术的安全性, 有效性尚不确切, 国内外并无相关临床研究和循证医学依据, 暂不宜应用于临床. 因此在2009年之后, 网戒中心使用的是G9805-C低频电子脉冲治疗仪, 但本质上还是电刺激治疗, 也就是所谓的电击. 在2016年又一次声讨临沂网戒中心的浪潮之后, 传闻网戒中心为了谨慎起见主动停用了电击, 使用了替代方案并加大了力度 (详见下文). 目前网戒中心使用电击的情况尚不明确.
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;3.1.1. 杨叔专场
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;针对某一位特定盟友, 由杨永信本人亲自实施的电击治疗. 进了杨叔专场的盟友通常会接受最为严厉的 “治疗”.有人控诉杨把针插入他们指甲盖与指头的缝隙中, 使用4台机器同时治疗, 把旋钮拧到最大频率, 还宣称机器数量 "没有上限" . "治疗" 对象是通常犯了严重错误的盟友, 或是杨认定的顽固分子, 是 "挑战杨叔模式" , 不能轻易放过.
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;3.1.2. 开门就进程序
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;只要十三号室开门使用, 进入该程序的 "盟友" 就要去 "治疗" . 早期说法为 "必修课" . 被纳入开门就进程序的盟友中曾不止一位因此寻死.
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;3.1.3. 全面程序
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;除开门就进程序外, 其他程序都要体验的一种全面惩罚程序.
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;3.1.4. 典型
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;严重违反某些规定后被特别关照, 定为 "XX典型" , 已知的分类有：兴奋典型, 消极典型, 坐姿典型, 军训典型, 日记课典型等等. 被定为典型会频繁进入十三号室, 相当于某个领域的 “老赖”.
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;3.1.5. 加圈制度
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;网戒中心实行圈机制, 违反规定就 "加圈" , 等于增加不良积分. 加圈的考核标准细分为心态, 卫生, 职责, 亢奋, 兴奋, 作业, 睡觉, 无聊等等（不同时期参数, 标准有变）. 每个类别对应不同的“圈”数. 点评师, 班委都有加圈权限. 例如, 在走廊上跑动属于“亢奋”, 1次亢奋加5个圈.
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;积分每周统计, 即 "周总结" (详见3.1.7), 每位盟友的圈数累积到规定上限, 就要进十三号室电击, 称之为 "兑现". 非睡觉时间不准躺或者半躺, 否则称为赖床, 要加圈, 上课不准打瞌睡, 要加圈. 坐不好要加圈, 说脏话加圈 (连靠和哇噻都是脏话), 二郎腿加圈, 被子不齐加圈, 名目非常繁多, 曾经也有人因为 "洁癖" 被加过圈.
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;而通过上报他人, 辩论课上得第一等途径, 经点评师点评后可以加相应的表扬圈以示表扬. 累积一定数量的表扬圈才可以出院 (详见4.2.2).
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;3.1.6. 激励
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;每天结束后, 就会统计出各自小组圈数最多的盟友, 一周内超过三次也需 "治疗". 属于额外, 附加的惩罚机制, 称之为 "激励".
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;3.1.7. 周总结
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;一周内加圈数超过规定上限的盟友要在每周四做 "治疗". 不同时期, 不同身份的盟友要求的圈数不同. 据2014年入院的一位盟友描述, 入院不到1个月的盟友累计60个圈就要 "兑现", 入院2-3个月是45个圈, 入院3-4个半月是30个圈, 入院超过4个半月是15个圈. 而班委累计10个圈就要做 "治疗". 就像游戏通关, 越到后期生存环境越严苛.
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;在传出禁止电击后也相应更改了规则, 即体罚. 例如信念纠偏体操 (详见3.2), 在网戒中心惩罚一次最少200个, 最多的一人被罚了30000个. 除此之外还有抱沙袋等等的体力惩罚, 饮食上只能吃清水煮萝卜白菜, 如果拒绝配合就直接绑到床上束缚 (详见3.4), 再不配合就送去 "针灸" (详见3.5).
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;3.1.8. 起飞
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;出现盟友不服从或其他突发的状况时, 接待员会使用一种快速控制的方式, 称之为 "起飞". 由五位每周训练的接待员, 分别抱住盟友的两条胳膊, 两条腿和腰, 头朝上, 四脚离地, 抬进十三号室, 动作迅捷, 一气呵成.

<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;3.1.9. 全员责任
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;盟友犯下严重错误, 诸如逃跑, 自杀等, 班委要承担连带责任, 集体被 "治疗". 据2014年入院的一位盟友说, 他住院期间有女孩数次自杀未遂, 他所在的班委班子每次都要承担连带责任. 一位入院一天的盟友深夜两三点从卫生间的窗户跳楼逃跑, 他们也为此承担了责任. 实际上全员责任的标准取决于杨本人, 当他感觉网戒中心 "氛围下降" 时, 同样全员责任.
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;3.2. 信念纠偏体操程序
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;除周日外, 每天需要做至少2小时的纠偏信念操. 体操动作取材自藏传佛教信仰者 "磕长头", 该操可分解为8个动作: 1. 站直身体, 双手笔直伸过头顶击掌; 2. 双手放在胸前合十; 3. 双膝下跪; 4. 放直身体五体投地; 5. 前臂与地面呈九十度抬起; 6. 放下前臂并用掌心拍击地面; 7. 收回双手支撑于地面; 8. 恢复直立. 因为运动强度大, 是盟友闻之膝软的一种严酷体罚.

<br/>&nbsp;&nbsp;&nbsp;&nbsp;3.3. 药物治疗
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;网戒中心会给盟友开各类不同的精神类药物, 造成不良反应明显, 主要有嗜睡, 精神不振, 浑身无力, 身体虚胖. 这给中心管制盟友提供了相当的便利, 也给了家长营养充足的假象.
<br/><p class="picbox"><a href="https://www.iaders.com/lywjzx/friends/4963/attachment/20181033/#main" rel="attachment wp-att-5134"><img class="aligncenter size-full wp-image-5134" src="https://raw.githubusercontent.com/ZjzMisaka/iaders/master/img/2018/07/20181033.jpg" alt="" width="720" height="407"></a></p>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;以账单中提到的药物为例:
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;3.3.1. 盐酸帕罗西汀片
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;根据警示语, 此类药物增加了儿童, 青少年和青年 (≤24岁) 患者自杀倾向 (自杀意念和自杀的行为) 的风险. 任何人如果考虑将盐酸帕罗西汀片或其他抗抑郁药物用于儿童, 青少年和青年 (≤24岁), 都必须在其风险和临床需求之间进行权衡.
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;常见 (≥1/100, &lt;1/10) 的不良反应分别有: 胆固醇水平升高, 食欲减退, 体重增加; 嗜睡, 失眠和兴奋, 异常的梦境 (包括梦魇); 眩晕, 震颤, 头痛, 情绪不稳定; 视力模糊; 高血压, 心动过速; 打哈欠; 便秘, 腹泻, 呕吐, 口干; 出汗, 瘙痒; 虚弱无力, 体重增加; 关节痛; 耳鸣; 停用药物后眩晕, 感觉障碍, 睡眠障碍, 焦虑, 头痛.
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;很常见 (≥1/10) 的不良反应分别有: 恶心; 性功能障碍.
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;在儿童的临床研究中, 至少有2%的患者报告有下述不良事件发生, 发生率至少为安慰剂的两倍：情绪不稳定 (包括自我伤害, 自杀念头, 自杀企图, 哭泣和情绪波动), 敌意, 食欲减退, 震颤, 出汗, 痉挛和激动. 自杀念头和自杀倾向主要见于患有重性抑郁症的青少年的临床试验中. 敌意多见于患有强迫性神经症的儿童, 特别是12岁以下的儿童.
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;3.3.2. 枸橼酸坦度螺酮胶囊
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;据国外报道, 调查总例数1451例中有150例 (10.3%) 出现不良反应及实验室检查值异常. 主要的不良反应有嗜睡43例 (3.0%), 步态蹒跚16例 (1.1%), 恶心13例 (0.9%), 倦怠感11例 (0.8%), 情绪不佳11例 (0.8%), 食欲下降10例 (0.7%), 肝功能异常, 黄疸 (&lt;0.1%). 主要实验室检查值异常有AST (GOT), ALT (GPT)升高.
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;3.3.3. 茴拉西坦胶囊
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;该药在网戒中心用以提升盟友记忆力.
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;不良反应可见口干, 食欲减退, 便秘, 头昏, 嗜睡, 停药后消失, 偶有兴奋, 躁动盒全身湿疹的报道.
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;3.4. 束缚
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;"束缚" 这一惩罚方式诞生在停用电击之前, 并在停用电击期间发扬光大. 盟友被布条牢牢绑在床上, 时间长达数星期, 任何理由都不得下床, 包括吃饭, 睡觉, 上厕所.

<br/>&nbsp;&nbsp;&nbsp;&nbsp;杨永信在盟友间有个外号, 叫 "木乃伊法老". 可想而知捆绑有多扎实.
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;3.5. 针灸
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;针灸被用作最终惩罚方, 而且 "接待" 也从盟友换成了成年人, 这也是为什么不能确定所谓 "针灸" 是不是低频脉冲的原因, 禁止盟友围观, 禁止私下讨论, 但每个在网戒中心的人都知道有这么一种 "针灸" , 很痛苦, 不过 "针灸" 确实用的少了.
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;3.6. 点评课
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;点评课是杨永信 "打开网瘾患者心灵世界" 的重要手段. 课堂上, 家长与 "盟友" 同时接受杨永信的点评, 剖析家庭问题.
<br/>&nbsp;&nbsp;&nbsp;&nbsp;一场成功的点评课, "在仇恨, 敌视, 排斥中开始, 在泪水, 拥抱, 微笑中结束".
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;在一场点评课中, 盟友们需要互相举报 (详见3.7), 自我晾晒 (详见3.8), 或者表演戒网瘾情景剧, 或者表达自己的悔恨之情与对网戒中心和杨永信的感恩之情, 等等. 杨永信的点评课的末尾, 盟友们往往要与家长和杨永信本人哭成一团, 直到下跪, 磕头, 最终在这种 "感人" 的气氛中结束.
<br/><p class="picbox"><a href="https://www.iaders.com/lywjzx/friends/4963/attachment/20181038/#main" rel="attachment wp-att-5139"><img class="aligncenter size-full wp-image-5139" src="https://raw.githubusercontent.com/ZjzMisaka/iaders/master/img/2018/07/20181038.jpg" alt="" width="720" height="314"></a></p>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;3.7. 上报制度
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;也叫 "问题反馈". 网戒中心中有各类完善, 便捷, 多样的举报渠道, 并且鼓励盟友对盟友与家长对盟友的举报, 通过举报, 可以相应地减少自己的圈数或增加表扬圈, 也可减轻自己的连带责任.
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;在网戒中心, 人人都不该有秘密, "上报" 有益于 "盟友" 改造. 根据一位盟友带出的 "心理导入本" 记载, 严重的如自杀嫌疑, 出逃嫌疑, 说方言, 伸脖子, 随意出入储藏室, 站军姿时男盟友看女盟友一眼, 休息时看着窗外 (被指有出逃想法), 吃药时男盟友跟女盟友说句话等等, 被上报后直接 "兑现". 如上文所说, 举报人可以是盟友, 也可以是父母.
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;通过上报制度可以最大限度地限制小团体的产生, 减少盟友间的信任, 减少盟友对家长的信任, 增加无助感.
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;3.8. 自我晾晒制度
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;盟友或家长公开自曝, 反省的行为, 方式包括盟友在日记中袒露曾犯下的错, 家长在点评课上陈述自己的过错等等. 网戒中心鼓励晾晒, 按照杨永信的说法, 自我晾晒是一个自我纠偏, 自我点评, 自我反省和大彻大悟的过程. 这也是 "永信模式" 独创的108个治疗环节之一.
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;在自我晾晒过程中, 盟友们为了展示自己的态度诚恳, 认错积极, 往往需要捏造自己的犯错经历, 甚至是自己的犯罪史, 或不堪的个人经历. 在自我晾晒的过程中, 盟友的自尊心会受到极大的摧毁. 这是杨永信达成把盟友训练成一个提线木偶的目的的一个必要的过程.

<br/>&nbsp;&nbsp;&nbsp;&nbsp;3.9. 网上作业
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;入院一个月以上的盟友, 要在网戒中心内部机房的电脑上完成将近几十万字的各项打字作业, 包括 "我的过去", "我的蜕变", "我离地狱有多远", "我的走偏" 等等, 每周一次, 在机房打, 出院上交. 相当于戒网 "大学" 的毕业论文. 在一个阶段未完成相应作业的会被电击, 网上作业未完成不允许出院.

<br/>&nbsp;&nbsp;&nbsp;&nbsp;巨大的作业量与无法出院的压力, 也成为了盟友们的一大负担.
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;3.10. 戒网瘾同盟班会制度
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;简称班委, 盟友中诞生的自我管理组织, 由全体盟友民主投票选举, 通常由点评师和家长首肯. 班委有给盟友加圈的权限, 且通常不会被电击, 在吃药, 行动自由, 自保等级等方面有相当的特权发挥空间. 不同时期班委体系有变更, 早期班委由班长和多位委员组成.
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;班委在班级中享有特权的同时, 也负有维持秩序, 管理盟友, 检举上报等责任. 在其他盟友犯下严重过错时, 班委将负有连带责任, 集体接受 "治疗".
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;在这种压力下, 班委在检举上报时不会留情, 但据说也有小团体共同推举班委的情况.
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;3.11. 家长同盟委员会制度
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;简称家委会, 是家长自我管理的组织. 家委会网戒中心维持运行的重要保障. 家长委员监督 "盟友" 日常行为, 管理中心秩序, 在一切事务上协助 "杨叔". "盟友" 入院时, 家长与家委会签订入院协议, 家长要保证按照 "相信, 配合, 坚持" 的方针. 家委会是名义上的管理者, 实质上听命于杨永信, 对其深信不疑.
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;家委会有独立的办公室, 其成员承担不同的责任, 例如上报盟友的不良行为, 防止外人进出网戒中心, 时刻贴身护卫杨永信, 或者作为别动队 (详见5.2) 抓捕盟友.

<br/>&nbsp;&nbsp;&nbsp;&nbsp;已出院盟友的家长也会保留家委身份, 成为院外家委. 网戒中心的家委遍布中国各地, 可以很方便的在异地抓捕盟友.
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;家委会对盟友造成的压力也是不容小觑的.
<br/>
<br/>4. 出院前
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;4.1. 再偏体验程序
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;针对即将出院, 处在 "黄金期" 的盟友 (一般指出院前一个月的时间). 为了防止出院后再偏, 提前体验再偏程序, 起威吓作用.
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;4.2. 出院条件判断
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;盟友需要达到一定条件才能被允许出院, 出院的盟友被称为 "精品". 为了成为 "精品" 早日出院, 盟友们必须做到绝对服从.
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;已知的条件有:
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;4.2.1. 疗程未满
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;未满一个疗程不许出院. 2015年8月入院的盟友说, 当时五个半月为一个完整疗程; 按战网魔的说法, 一个疗程为四个月.
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;4.2.2. 表扬圈未够数
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;在实行表扬圈机制的时期, 累积一定数量的表扬圈才可以出院. 不过有替代方案即以一个圈100元的方式补足表扬圈.
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;4.2.3. 网上作业未完成
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;网上作业未完成者不许出院, 直到完成作业为止.
<br/>
<br/>5. 出院后
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;5.1. 长效机制
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;长效机制确保出院盟友行为, 状态符合规范的机制. 对盟友来说, 这是网戒中心内部恐怖统治在院外的延续.
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;在院外的盟友大多被要求继续遵守在院内的规则, 保持在院内的习惯, 按时做信念纠偏体操, 甚至服药. 盟友要牢牢地被掌控在父母手中, 否则很有可能被定性为再偏, 抓回中心继续治疗.
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;最新的机制里包括分享, 充电, 深造, 再偏四种情况.
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;5.1.1. 分享
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;出院盟友每周定期返院上点评课一天, 收费, 分享自己的改造心路历程, 后按照家长和杨永信的满意程度定性, 满意则可回家.
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;5.1.2. 充电
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;家长和杨永信对出院盟友的表现有轻微不满, 需至少再次入院一周.
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;5.1.3. 深造
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;家长和杨永信对出院盟友的表现比较不满, 返院深造, 至少入院一个月.
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;5.1.4. 再偏
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;家长和杨永信对出院盟友的表现非常不满, 则至少入院三个月. 并会进入 "开门就进" 程序 (详见3.1.2), 这是长效机制中最严酷的一种. 再偏盟友胸前佩戴黑色铭牌, 上面写着 "再偏 / 知耻". 有不少 "盟友" 因为拒绝被抓回网戒中心, 跳楼跳窗, 曾有人身亡.
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;5.2. 别动队制度
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;别动队是负责抓捕逃跑的盟友和强制再偏盟友返院的组织. 别动队由家委会成员组成, 一般是强壮有力的男性, 包括一些退伍军人身份或地头蛇混混身份的家长. 别动队中也会有数位盟友作为帮手. 别动队出动需收费.
<br/><figure><a href="https://www.iaders.com/lywjzx/friends/4963/attachment/%e5%88%ab%e5%8a%a8%e9%98%9f1/#main" rel="attachment wp-att-5124"><img class="aligncenter size-full wp-image-5124" src="https://raw.githubusercontent.com/ZjzMisaka/iaders/master/img/2018/07/别动队1.png" alt="" width="1439" height="1080"></a> <a href="https://www.iaders.com/lywjzx/friends/4963/attachment/%e5%88%ab%e5%8a%a8%e9%98%9f2/#main" rel="attachment wp-att-5125"><img class="aligncenter size-full wp-image-5125" src="https://raw.githubusercontent.com/ZjzMisaka/iaders/master/img/2018/07/别动队2.png" alt="" width="1437" height="1080"></a></figure>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;很多盟友没有选择出逃, 正是因为有别动队的存在. 无论逃到哪里, 最终都会被抓回来. 别动队可谓是控制盟友的一大法宝, 可以根绝很多盟友反抗或逃跑的念头.
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;5.3. 连续再偏免费治疗
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;如果一个盟友再偏了四次, 那么第四次治疗将会免费. 免费治疗减轻了家长的负担并鼓励家长将 "再偏" 孩子送回. 杨永信保证治到家长满意为止, 而对于盟友来说, 屡次再偏一次比一次严厉的 "治疗" 却是不小的震慑.
<br/><p class="picbox"><a href="https://www.iaders.com/lywjzx/friends/4963/attachment/201810316/#main" rel="attachment wp-att-5148"><img class="aligncenter size-full wp-image-5148" src="https://raw.githubusercontent.com/ZjzMisaka/iaders/master/img/2018/07/201810316.jpg" alt="" width="360" height="640"></a></p>
<br/>6. 再偏
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;6.1. 再偏程序
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;针对再偏返院的盟友. 每周至少电击一次, 周总结累计15个圈还要再电击一次. 该程序至少持续一个月, 同时包含特餐程序, 纠偏信念操程序
<br/>(详见3.2) 等等.

<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;6.1.1. 特餐程序
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;专治挑食行为, 一日三餐只能吃白水煮白菜豆腐. 也许中心就是靠这些来训练盟友逆来顺受的服从性的.
