世卫组织首次定“游戏障碍”为疾病》，值得注意的是，该文在篇首特别注明：<br/><br/><b>全球24位学者上书联名信，认为“为时过早、远未成熟”，但只得到了世卫组织轻描淡写的回复：“别担心”。</b><br/><br/><b>草案制定的过程中受到来自亚洲国家的巨大压力，一些国家迫切希望将游戏障碍纳入精神疾病范畴。</b><br/><br/><b>把游戏障碍列为精神疾病，可能会掩盖背后的真实原因，比如抑郁、焦虑或者其他心理问题。</b><br/><br/><b>外界最大担忧在于，打着“治网瘾”旗号的“杨永信们”会不会更理直气壮？”</b><br/><br/>综观以上文章，表面上看似乎是客观描述了对世卫组织决定的争议，对网瘾戒治是否科学存在担忧，整个一副忧国忧民的样子。<br/><br/>但，稍加分析，便不难看出其中明显的利益倾向。<br/><br/>其一，戒网瘾比网瘾更可怕。南方都市报文章的标题利益倾向就十分明显――“游戏成瘾再成话题：马化腾说腾讯被黑，我们更担心杨永信卷土重来”：一个担心游戏被黑，一个担心杨永信卷土重来，利益诉求无缝对接，一个“我们”更直接把报纸定位为游戏产业的利益代言！<br/><br/>在他们的眼里，杨永信，一个倾心于网瘾戒治事业，12年间成功救治6000余网瘾问题孩子，让6000余患难家庭重拾幸福，并因此获得国务院特殊政府津贴的优秀专家，被黑成摧残网瘾青少年的魔鬼，居然比让无数青春凋落的网瘾网游更可怕？！<br/><br/>其二，无视网瘾危害，虚无网瘾概念，试图重挑争议，对世卫组织决定进行翻案。南方周末文章《全球博弈“网游精神病” 世卫组织首次定“游戏障碍”为疾病》，不仅标题立意非常明显，而且篇首的特别注明表述的更加充分，尤其是“外界最大担忧在于，打着“治网瘾”旗号的“杨永信们”会不会更理直气壮？”这句话，更是把世卫组织决定连带后果的“可怕”都预言了，而且夸张到“最大担忧”的程度！<br/><br/>而事实上，真实的杨永信和临沂网戒中心并非如此。<b>无论是世卫组织还是杨永信本人，从未忽略过网瘾“背后的真实原因，比如抑郁、焦虑或者其他心理问题。”</b><br/><br/>在此，我们不妨再科普一下：<br/><br/><b>关于临沂网戒中心和其创办人杨永信的事实真相：</b><br/><br/><b>山东省临沂网戒中心</b>：创办于2006年1月，是三级甲等精神专科医院临沂市精神卫生中心的下属特色科室。<br/><br/><b>创办人杨永信</b>：是该院副院长，享受国务院政府特殊津贴的精神卫生专家，出于<b>对网络成瘾及问题孩子、问题家庭与性格缺陷等问题的关联研究</b>，他创立了一套集医院、学校、家庭、军营特色于一体，集网络成瘾戒治、性格缺陷矫治、问题家庭诊疗、社会功能康复等功能于一身的综合干预治疗模式。该模式包含108个环节，其核心环节就是以社会主义核心价值观为内核，以优秀的国学传统为基本依托，以理想信念培养为目标的心理点评。同时包含了大量的辅助性环节：如意志训练，习惯养成，感恩教育，社会功能康复训练（如阅读、写作、演讲、才艺展示、亲子交流、内务整理等）等。因为其戒治模式的综合性，前来求治的患者除深度网瘾患者之外，普通精神疾病患者、社会功能缺乏者、性格缺陷者、h赌涉毒涉黄涉黑者皆有之。十几年间，让六千余患者成功回归家庭学校并走向社会，有效预防和遏止了青少年犯罪，且从未出现一例医疗安全事故，大量成功的案例验证了模式的安全科学有效，也为他赢得了家长的认可和政府的嘉奖。<br/><br/>2016年4月25日，该成果通过了由临沂市科技局组织、由中国工程院院士领衔的业内高端专家组的鉴定，结论为：<b>“该模式能够很好地解决长期困扰网瘾戒治方面的理论、技术和临床等重大问题，具有理论技术指导、操作安全可行和临床高效稳定等特性。同时，杨氏网戒模式集教育学、医学、心理学等于一体，形成了缜密、前沿、高效可行的科学系统，可广泛应用于网瘾、酒瘾、毒瘾、赌瘾、药物成瘾等各种瘾症的戒治，用于其他各种心理精神疾病的治疗和社会功能康复，挽救问题青少年，有效干预其危险行为及预防违法犯罪。开辟了多学科、多领域综合应用的成功典范，其综合技术填补国内空白，在国际上具有显著创新性，居国际先进水平</b>”。<br/><p
